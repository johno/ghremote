# ghremote

Git remote shortcut. A bash script bundled with npm.

## Installation

```
$ npm i -g ghremote
```

## Usage

```sh
$ ghremote
GitHub remote shortcut


Usage:
  $ ghremote <username>/<repo>


Example:
  $ ghremote johnotander/pixyll
  $ ghremote upstream johnotander/pixyll
```
